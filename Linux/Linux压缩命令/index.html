    Linux压缩以及解压文件命令

    Ubuntu上解压压缩文件的几种解压方法¶

    .7z文件¶

    参考:linux 解压 7z 乱码,7z-linux下解决中文名乱码的终极办法

    准备工作

    打开终端，在终端中输入 

    1
    wget http://netix.dl.sourceforge.net/project/p7zip/p7zip/15.09/p7zip_15.09_src_all.tar.bz2

    此时会自动下载7z的源码，你的终端在什么地方打开的，那么源码就下载到什么位置，但是如果你的终端是在桌面上打开的，那么7z的源码则下载至家目录里面的。

    源码下载完成之后解压该源码，在终端中输入 tar -jxvf p7zip_15.09_src_all.tar.bz2 ,这样源码在什么地方，则解压就解压到什么地方。

    如果你想解压到别的地方，则可以输入 tar -jxvf p7zip_15.09_src_all.tar.bz2 -C /xx/xxx/xxx ,-C大写，后面接要解压的路径

    解压后得到一个名为 p7zip_15.09 的文件夹，进入源码文件夹里面，终端中输入 cd p7zip_15.09 即可进入到源码文件夹里面，然后运行代码 sudo make install ，该代码是用来编译源码的，编译完成之后就可以使用解压7zip了

    如果编译失败，那么可以尝试自己安装，输入命令 sudo apt install p7zip 即可，这样系统就会自动安装p7zip了

    如果是CentOS系统，可以源码编译，也可以使用yum命令来安装，这种适合CentOS 7及以上的系统

    在终端中输入 sudo yum install -y p7zip 即可下载

    如果编译的时候提示gcc命令未找到时，则输入以下两个命令

    sudo yum -y update gcc 和 sudo yum -y install gcc+ gcc-c++

    举个例子：比如我要解压一个名为 text.7z 的压缩包，那么只需要在该压缩包的文件目录里面打开终端，输入命令 7za x test.7z ，这样 text.7z 就解压成一个文件夹了，如果你想把 text.7z 压缩包解压到别的文件夹下，则可以输入 7za x test.7z -o/解压路径

    7z命令中的参数解释说明

    7za a ：向压缩包里添加文件或创建压缩包，如向001.7z添加001.jpg，执行：7za a 001.7z 001.jpg；将001目录打包执行：7za a 001.7z 001；

    7za d ：从压缩里删除文件，如将001.7z里的001.jpg删除，执行：7za d 001.7z 001.jpg

    7za l ：列出压缩包里的文件，如列出001.7z里的文件，执行：7za l 001.7z

    7za e ：解压到当前目录，目录结构会被破坏，如001.rar内有如下目录及文件123/456/789.html，执行：7za e 001.rar，目录123和456及文件789.html都会存放在当前目录下。

    7za x ：以完整路径解压

    但常用的也只有解压和压缩这两个命令

    .zip文件¶

    参考ubuntu下解压zip文件出现中文乱码

    参考Ubuntu等linux系统时经常会遇到解压压缩文件出现乱码，之所以这样是因为由于zip格式并没有指定编码格式，Windows下生成的zip文件中的编码是GBK/GB2312等，因此，导致这些zip文件在Linux下解压时出现乱码问题，因为Linux下的默认编码是UTF8。

    zip的处理方式主要有以下两种

    1.unzip¶

    解压时指定字符编码(如果你的zip压缩文件是在Windows上压缩的，那么在Linux上则要指定编码解压，否则会出现解压的文件会有乱码的情况)，比如我要解压 text.zip ，只需在终端中输入以下代码

    1
    unzip -O GBK text.zip

    2.unar¶

    首先要下载unar，在终端中输入 sudo apt install unar ,等待系统下载完

    比如我要解压 text.zip ,只需在终端中输入以下代码

    1
    unar text.zip

    .RAR文件¶

    参考在 Ubuntu Linux 中使用 RAR 文件

    解压¶

    首先在终端中输入 sudo apt-get install unrar 下载unrar命令行工具，比如我要解压一个名为 text.rar 的压缩包，只需要在该目录下打开终端，输入 unrar x text.rar 即可解压
